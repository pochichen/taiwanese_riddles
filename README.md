# 台灣趣味猜謎 (Taiwanese Riddles)

一個簡單、直覺且支援鍵盤操作的台灣趣味猜謎 Web App。收錄了各種傳統與現代的謎語，讓你在輕鬆互動中體驗猜謎的樂趣！

## 開始遊戲

請直接點擊下方連結開始遊戲：
**[開始猜謎遊戲](https://pochichen.github.io/taiwanese_riddles/)**

---

## 操作說明

本 App 針對電腦使用者設計了極簡的「單鍵操作」邏輯，也支援行動裝置點擊。

### ⌨️ 鍵盤操作（推薦）
- **[空白鍵 Space]**：
  1. 第一次按下：顯示題目**提示**（若有）。
  2. 第二次按下：顯示**正確答案**。
  3. 第三次按下：自動跳轉至**下一題**。
- **[左/右方向鍵]**：切換上一題或下一題。

### 畫面操作
- 點擊左上/右上按鈕切換題目。
- 點擊左下 ** 提示** 按鈕查看提示。
- 點擊右下 ** 看答案** 按鈕顯示答案。

## 技術架構

- **前端**: 原生 JavaScript (Vanilla JS), HTML5, CSS3。
- **資料儲存**: JSON 格式。
- **爬蟲工具**: 使用 Python (BeautifulSoup) 從教育機構網站自動化擷取並整理謎語資料。
- **部署**: GitHub Pages。
